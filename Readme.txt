Underpipies 1.0.0
===================

Version 1.0.0 was released March 24, 2020, was tested using Factorio v0.17.79, and was authored by Nexusmaniac, based solely on the Underneathies mod.

This mod changes the names of Underground Pipes into "Underpipies". That's it. No more, no less.
